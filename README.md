# 🏢 AR Interactive Building Visualization

> An Augmented Reality application developed using **Unity** and **Vuforia Engine** for visualizing 3D building models in a real-world environment.

---

## 📌 Project Overview

AR Interactive Building Visualization is an Augmented Reality (AR) application that allows users to visualize a 3D building model by scanning an image target using an Android device. The project demonstrates the use of AR technology to improve architectural visualization through an immersive and interactive experience.

---

## 🎯 Objectives

- Develop an Augmented Reality application for building visualization.
- Display a 3D building model using image target recognition.
- Enable users to interact with the building through 360° rotation.
- Enhance architectural presentation using AR technology.

## 🚀 How to Run

1. Open the project in Unity.
2. Import the Vuforia Engine package.
3. Configure the Vuforia License Key.
4. Build the project for Android.
5. Install the APK on an Android device.
6. Scan the image target to visualize the 3D building.
